# Gym Timer (Web版)

筋トレのインターバルを自動で管理するWebアプリです。
React Native版の開発前に試作したプロトタイプです。

## 主な機能

- 種目・セット数・インターバルの登録と保存
- 種目終了後に次の種目へ自動遷移し、タイマーをリセット
- データ永続化

## 使用技術

- HTML / CSS / JavaScript
- localStorage（データ保存）
- Wake Lock API（画面オフ防止）

## 開発期間

2026年3月
